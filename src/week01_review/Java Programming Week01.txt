Basic Shortcuts:
    Mac:
        copy ==> command + c
        paste ==> command + v
        undo ===> command + z
        cut ===>  command + x
        select all ===> command + A
        save ===> command + s
        screenshot ===> command + shift + 3


    Windows:
        copy ==> cntrl + c
        paste ==> cntrl + v
        undo ===> cntrl + z
        cut ===>  cntrl + x
        select all ===> cntrl + A
        save ===> cntrl + s
        screenshot ===> windows key + prtscn (will be saved under pictures directury)
                         alt + prtscn
